# rails-consultancy

## how to get started
``` 
#if ocurred an error with webpack use 'rails webpacker:install'
rails db:setup
rails db:create
rails db:migrate
#if not migrated on test db, use 'rails db:migrate RAILS_ENV=test'
rails s
```
I'm trying to use DDD, OBS: model with ActivityRecord will be used as DAO.
